If someone suddenly asked you to make a tree, you might feel very confused.

To get ready for such a day, I've made the simplest tree, a binary tree, and organized the basics needed for development.

## Ideas About Binary Trees

1. A binary tree is made up of a group of nodes.
2. The first value is set as the root node.
3. Values smaller than the root node go to the left child node, and bigger values go to the right child node. This is done over and over.
4. We can visit all nodes in a binary tree through in-order traversal.

   > In-order traversal: Left child -> Current node -> Right child
   >
   > [https://www.geeksforgeeks.org/inorder-traversal-of-binary-tree](https://www.geeksforgeeks.org/inorder-traversal-of-binary-tree/)

## Node Class

First, we need to define the node that makes up the tree.

In the constructor, we set the value of the root node and set up the left and right child nodes.

```typescript
// Define Node class
class Node {
  value: number;
  left: Node | null;
  right: Node | null;

  constructor(value: number) {
    this.value = value;
    this.left = null;
    this.right = null;
  }
}
```

## Node Insert Function

```typescript
// Function to insert a new node into the binary search tree
function insert(root: Node | null, key: number): Node {
  // If the tree is empty, return a new node
  if (!root) {
    return new Node(key);
  }

  // Compare the current node's value with the value to insert and move to left or right child
  if (key < root.value) {
    root.left = insert(root.left, key);
  } else {
    root.right = insert(root.right, key);
  }
  return root;
}
```

## In-order Traversal Function

```typescript
// Function to print the tree's nodes through in-order traversal
function inorderTraversal(root: Node | null): void {
  if (root) {
    // First, traverse the left child
    inorderTraversal(root.left);
    // Print the current node's value
    console.log(root.value);
    // Traverse the right child
    inorderTraversal(root.right);
  }
}
```

## Usage Example

```typescript
let root: Node | null = new Node(10);
root = insert(root, 20);
root = insert(root, 5);
root = insert(root, 15);

console.log("In-order traversal result:");
inorderTraversal(root); // 5, 10, 15, 20
```
