---
title: "When to use which naming convention?"
date: "250204"
tags: ["Design"]
---

| Naming Convention    | Common Usage                                                  |
| -------------------- | ------------------------------------------------------------- |
| `camelCase`          | Variables, functions (JavaScript, TypeScript, Java, C#)       |
| `snake_case`         | Variables, functions (Python, databases)                      |
| `kebab-case`         | URLs, CSS class names, file names                             |
| `PascalCase`         | Classes, types, interfaces (JavaScript, TypeScript, Java, C#) |
| `UPPER_SNAKE_CASE`   | Constants, environment variables                              |
| `Hungarian Notation` | Legacy code (rarely used)                                     |
