---
title: 'How to Change JetBrains Shortcuts'
date: '250611'
tags: ['Tool']
---

1. Open the Settings window

   - Windows/Linux: `Ctrl + Alt + S`
   - macOS: `⌘ + ,`

2. Select Keymap from the left-hand menu.

3. In the search bar on the right, type the name of the action you want to change. Examples:

   - Move Caret
   - Execute Statement
   - Submit Changes

4. Right-click the desired command → Select Add Keyboard Shortcut.

5. Enter your desired key combination and click OK.

   - To use multi-stroke shortcuts, check the Second stroke option.

6. If there is a conflict with existing shortcuts, a warning will appear. Click Modify Shortcuts to resolve it.

7. Once you're done, click Apply/OK to save the changes.
