지난 포스팅에서 갤럽모드를 통해 팀소트를 더 빠르게 최적화할 수 있다고 했는데, 이번에는 갤럽 모드에 대해 알아보겠습니다.

#### Tim Sort 간단 리마인드

먼저 Tim Sort가 어떻게 작동하는지 간단히 정리해보겠습니다.

1. Tim Sort는 배열을 작은 부분으로 나눕니다.
2. 이 작은 부분들을 정렬합니다.
3. 그리고 이 정렬된 부분들을 병합합니다.

이때, 병합 과정에서 갤럽 모드를 사용하면 더 빠르게 병합할 수 있습니다.

#### 갤럽(gallop) 모드 예시

두 개의 정렬된 배열 A와 B가 있다고 가정해 봅시다.

- A: [1, 3, 5, 7, 9]
- B: [10, 20, 30, 40, 50]

이 두 배열을 병합할 때, 일반적인 방법은 A와 B의 요소를 하나씩 비교하면서 병합하는 것입니다. 하지만 이 경우, A의 모든 요소가 B의 모든 요소보다 작습니다.

Galloping mode는 이런 상황을 빠르게 파악합니다.

1. 먼저 B[0]과 A[0]를 비교합니다. (10 > 1)
2. 그 다음 B[0]과 A[1]을 비교합니다. (10 > 3)
3. 그 다음은 A[3]과 비교합니다. (10 > 7)
4. 마지막으로 A[7]과 비교하려 하지만, A의 끝에 도달했습니다.

이렇게 빠르게 건너뛰면서 비교하는 것이 갤럽모드의 핵심입니다. 말 그대로 "질주(gallop)"하듯이 빠르게 움직이는 것이죠.

만약 목표값보다 큰 요소를 찾으면, 마지막으로 확인한 두 지점 사이에서 이진 검색을 수행합니다.

#### 갤럽 모드 장점

1. 비교 횟수가 크게 줄어듭니다.
2. 특히 한 배열의 모든 요소가 다른 배열의 모든 요소보다 크거나 작을 때 매우 효율적입니다.
3. 부분적으로 정렬된 큰 데이터셋에서 효과적입니다.

### 갤럽 모드 구현

기존 Tim Sort 코드에 갤럽 모드를 추가해봤습니다.

```ts
const INSERTION_SORT_THRESHOLD = 64;
const MIN_GALLOP = 7;

const timSort = (arr: readonly number[]): number[] => {
  const n = arr.length;
  let sortedArr = [...arr];

  for (let i = 0; i < n; i += INSERTION_SORT_THRESHOLD) {
    const end = Math.min(i + INSERTION_SORT_THRESHOLD - 1, n - 1);
    sortedArr = insertionSort(sortedArr, i, end);
  }

  for (let size = INSERTION_SORT_THRESHOLD; size < n; size = 2 * size) {
    for (let start = 0; start < n; start += 2 * size) {
      const mid = start + size - 1;
      const end = Math.min(start + 2 * size - 1, n - 1);
      if (mid < end) {
        sortedArr = merge(sortedArr, start, mid, end);
      }
    }
  }

  return sortedArr;
};

const insertionSort = (
  arr: readonly number[],
  left: number,
  right: number
): number[] => {
  const result = [...arr];
  for (let i = left + 1; i <= right; i++) {
    const key = result[i];
    let j = i - 1;
    while (j >= left && result[j] > key) {
      result[j + 1] = result[j];
      j--;
    }
    result[j + 1] = key;
  }
  return result;
};

const merge = (
  arr: readonly number[],
  start: number,
  mid: number,
  end: number
): number[] => {
  const result = [...arr];
  const leftArr = result.slice(start, mid + 1);
  const rightArr = result.slice(mid + 1, end + 1);

  let i = 0,
    j = 0,
    k = start;
  let leftWins = 0,
    rightWins = 0;

  while (i < leftArr.length && j < rightArr.length) {
    if (leftArr[i] <= rightArr[j]) {
      result[k++] = leftArr[i++];
      leftWins++;
      rightWins = 0;
    } else {
      result[k++] = rightArr[j++];
      rightWins++;
      leftWins = 0;
    }

    if (leftWins >= MIN_GALLOP || rightWins >= MIN_GALLOP) {
      while (i < leftArr.length && j < rightArr.length) {
        if (leftWins >= MIN_GALLOP) {
          const [newJ, newResult] = gallopRight(
            leftArr[i],
            rightArr,
            j,
            rightArr.length,
            result,
            k
          );
          j = newJ;
          k += j;
          result[k++] = leftArr[i++];
          leftWins++;
          rightWins = 0;
        } else {
          const [newI, newResult] = gallopRight(
            rightArr[j],
            leftArr,
            i,
            leftArr.length,
            result,
            k
          );
          i = newI;
          k += i;
          result[k++] = rightArr[j++];
          rightWins++;
          leftWins = 0;
        }

        if (leftWins < MIN_GALLOP && rightWins < MIN_GALLOP) {
          break;
        }
      }
    }
  }

  while (i < leftArr.length) {
    result[k++] = leftArr[i++];
  }

  while (j < rightArr.length) {
    result[k++] = rightArr[j++];
  }

  return result;
};

const gallopRight = (
  key: number,
  arr: readonly number[],
  start: number,
  end: number,
  result: number[],
  resultStart: number
): [number, number[]] => {
  let offset = 1;
  let lastOffset = 0;
  if (key > arr[start]) {
    const maxOffset = end - start;
    while (offset < maxOffset && key > arr[start + offset]) {
      lastOffset = offset;
      offset = (offset << 1) + 1;
      if (offset <= 0) {
        offset = maxOffset;
      }
    }
    if (offset > maxOffset) {
      offset = maxOffset;
    }
    lastOffset += start;
    offset += start;
  } else {
    return [start, result];
  }
  lastOffset++;
  while (lastOffset < offset) {
    const m = lastOffset + ((offset - lastOffset) >>> 1);
    if (key > arr[m]) {
      lastOffset = m + 1;
    } else {
      offset = m;
    }
  }
  for (let i = start; i < offset; i++) {
    result[resultStart + i - start] = arr[i];
  }
  return [offset, result];
};
```

### 마치며

팀소트는 상황에 따라 이 갤럽 모드를 사용할지 말지를 결정합니다. 효과가 없다고 판단되면 일반적인 병합 방식으로 돌아갑니다.

실제 팀소트 구현에서 갤럽 모드는 다양한 데이터 패턴에 대해 효율적으로 작동할 수 있게 해주는 중요한 기능을 합니다.
