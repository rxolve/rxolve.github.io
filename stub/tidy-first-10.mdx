> 정리하며 책의 문장을 요약해서 옮깁니다.

- 루틴에서 다루고 있는 일부 데이터가 명시적으로 전달되지 않는 것을 발견했습니다. 이련 경우 입력을 명확하게 하려면 어떻게 해야 할까요?
  - 루틴을 나누세요. 앞부분에서 매개변수 값을 채운 후, 뒷부분에서 명시적으로 전달합니다.
- 매개변수가 블록으로 전달되는 경우가 흔합니다.
  - (자바스크립트에서는 매개변수로 객체를 전달하는 경우입니다.)
  - 이렇게 하면 코드를 읽으면서도 어떤 데이터가 필요한지 알기 어렵습니다.
  - 매개변수를 변경하여 암묵적으로 사용하는 일과 같은 끔찍한 남용의 길이 열립니다.
- Before

  ```typescript
    params = {a:1, b:2}
    foo(params)

    function foo(params)
      ...params.a... ...params.b...

  ```

  - After

  ```typescript
  function foo(params)
    foo_body(params.a, params.b)

  function foo_body(a, b)
    ...a... ...b...

  ```

- 명시적인 매개변수가 필요한 또 다른 경우는 중괄호가 여러 개로 둘러싸인 코드에서 환경변수를 사용하는 경우입니다.
- 매개변수를 명시적으로 드러나게 만든 다음, 함수를 연쇄적으로 호출할 수 있게 준비하세요.
- 이렇게 만들면 코드는 읽기와 테스트, 분석이 쉬워집니다.
