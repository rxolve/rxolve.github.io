글을 쓰는 속도는 문장을 어떻게 만드느냐에 따라 달라진다.

```
색상 RGB 값을 각각 사용하기 때문에 입력 데이터는 3차원 벡터다.
```

이 문장의 주어는 '입력 데이터'다. 따라서 주어를 문장의 앞으로 뺄 수 있다.

```
입력 데이터는 색상 RGB 값을 각각 사용하기 때문에 3차원 벡터다.
```

이 문장은 인과관계가 있는 복문이므로 두 문장으로 나눌 수 있다.

```
입력 데이터는 색상 RGB 값을 각각 사용한다. 따라서 입력 데이터는 3차원 벡터다.
```

아는 내용을 바로 쓴다.

```
입력 데이터는 3차원 벡터다.
```

이제 입력 데이터가 3차원 벡터인 이유를 설명한다.

그래픽 프로그램에 익숙하지 않은 사람을 위해 부가 설명을 추가할 수도 있다.

```
입력 데이터는 3차원 벡터다. 색상 RGB 값을 각각 사용하기 때문이다.
```

간단한 문장 구조로 핵심만 말한 뒤, 필요에 따라 부가 설명을 하면 문장을 쉽게 쓸 수 있다.

이때 첫 문장의 주어를 가져다 소제목으로 만들면 자연스럽게 문단을 구성할 수 있다.
