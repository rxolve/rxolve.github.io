---

- **generateMetadata** 함수로 NextJS 에서 페이지별 메타데이터를 동적으로 설정할 수 있습니다.
- generateStaticParams 와 동일하게 서버 사이드에서 실행됩니다.
- 페이지 컴포넌트에서 params 를 인자로 받아 페이지에 전달되는 매개변수에 접근할 수 있습니다.
- 또한 API 호출을 통해 데이터를 가져와 메타데이터를 설정할 수도 있습니다.

  ```typescript
  export const generateMetadata = async ({ params }: TagDatePageProps) => {
    const { tag, date } = params;

    const post = await getPost(date);

    return {
      title: post.title,
      description: post.content,
      keywords: [post.title, tag],
      openGraph: {
        title: post.title,
        description: post.content,
        images: [
          {
            url: "https://rxolve.github.io/og-image.png",
          },
        ],
      },
    };
  };
  ```

  - 페이지들의 접근성이 좋아져서 기쁩니다.
