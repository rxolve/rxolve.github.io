NestJS 서비스나 DTO 변환 과정에서 범용적으로 사용할 수 있는 문자열 공백 제거 유틸리티 함수입니다.  
개행(`\n`), 탭(`\t`), 여러 개의 연속된 공백(`\s+`) 등을 모두 하나의 공백 또는 빈 문자열로 치환합니다.

#### 공백 제거 유틸 함수

```typescript
export const trimAllSpaces = (input: string, replaceWith = ""): string => {
  return input.replace(/\s+/g, replaceWith).trim();
};
```

#### 사용 예시

```typescript
const str = `  안녕하세요,   
    이런저런    프로젝트입니다.\t\t테스트 중입니다. `;

console.log(trimAllSpaces(str));
// Output: "안녕하세요,이런저런프로젝트입니다.테스트중입니다."

console.log(trimAllSpaces(str, " "));
// Output: "안녕하세요, 이런저런 프로젝트입니다. 테스트 중입니다."
```

#### 설명

- `\s+` : 모든 공백 문자(스페이스, 개행, 탭 등)를 한 번에 매칭.
- `replaceWith` : 기본적으로 `''`(빈 문자열)로 처리하지만, 필요하면 `' '`(하나의 공백)으로 변환 가능.
- `trim()` : 앞뒤 공백 제거.
