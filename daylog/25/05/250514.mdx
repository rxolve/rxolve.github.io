AWS Bedrock는 여러 AI 제공업체의 고성능 Foundation Model(FM)을 단일 API로 손쉽게 활용할 수 있는 완전 관리형 서버리스 서비스입니다.  
Bedrock을 통해 개발자는 모델 선택, 프롬프트 실험, 사내 데이터로의 맞춤화(파인튜닝 및 RAG), 에이전트 기반 자동화 기능을 통합해 Generative AI 애플리케이션을 빠르게 구축할 수 있습니다.  
이 서비스는 AWS의 기존 보안·프라이버시·책임 있는 AI 가이드라인을 준수하며, 비용은 사용량 기반으로 지불하는 구조를 가집니다.

## 서비스 개요

#### 완전 관리형 서버리스 경험

- Bedrock은 인프라 관리가 전혀 필요 없는 서버리스 경험을 제공합니다. 마치 전기 콘센트에 플러그만 꽂으면 불이 켜지는 전등처럼, 모델을 호출하기만 하면 바로 AI 추론을 시작할 수 있습니다.
- AWS 콘솔의 텍스트·이미지·채팅 플레이그라운드를 통해 GUI 기반 실험도 가능하며, 준비가 되면 `InvokeModel` API로 손쉽게 전환할 수 있습니다.

#### 다양한 Foundation Model 선택

- Anthropic, Cohere, AI21, Stability AI, Meta, Amazon Titan 등 최상위급 FM을 단일 엔드포인트에서 선택할 수 있습니다.
- Bedrock Marketplace에서는 100개 이상의 인기·전문 FM을 탐색·테스트·사용할 수 있어, 마치 AI 모델 쇼핑몰에서 필요한 상품을 골라 담듯 유연한 선택이 가능합니다.

## 주요 기능

#### 맞춤화(Fine-Tuning & RAG)

- 파인튜닝: 자체 라벨 데이터로 원본 모델의 복사본을 생성해 개인 전용 모델을 만들 수 있으며, 원본 FM에는 데이터가 유입되지 않습니다.
- Retrieval-Augmented Generation(RAG): 사내 지식 저장소에서 관련 문서를 꺼내어 프롬프트에 주입하는 방식으로 최신 정보를 반영하는 ‘지식 주입형’ 응답을 생성합니다.
- Knowledge Bases for Bedrock: 완전 관리형 RAG 기능으로, 개발자는 번거로운 백엔드 작업 없이도 회사 데이터로 즉시 커스텀 응답을 구현할 수 있습니다.

### 2.2. 에이전트(Agents)

- Bedrock 에이전트는 기업 시스템·외부 API를 호출해 업무(Task)를 자동화하는 작은 로봇과 같습니다.
- 예컨대, 고객 지원용 챗봇이 CRM 시스템에서 주문 상태를 조회해 답변하거나, 재고 시스템과 연동해 실시간 재고 알림을 제공하는 시나리오를 손쉽게 구성할 수 있습니다.

### 2.3. 보안·책임 있는 AI

- AWS의 IAM, VPC 엔드포인트, KMS 암호화 등 전통적 보안 기능을 그대로 활용할 수 있습니다.
- Bedrock은 책임 있는 AI(Responsible AI) 프레임워크를 지원해, 프롬프트 필터링·출력 검증·거버넌스 로깅을 통해 윤리적·규제 준수가 가능합니다.

## 가격 모델

- Bedrock는 실제로 호출한 추론 요청과 데이터 전송량에 대해서만 비용을 지불하는 종량제(Pay-as-you-go) 구조입니다. 마치 수도·전기를 사용한 양만큼만 고지서를 받는 것과 유사합니다.
- 모델 종류에 따라 요청당 비용이 상이하며, 파인튜닝·RAG용 저장소 사용에도 별도 요금이 발생할 수 있습니다. 요금 계산기는 AWS 요금 페이지에서 직접 시뮬레이션해볼 수 있습니다.

## 최신 업데이트

- 2024년 4월: 모델 평가(Model Evaluation) 기능이 일반에 공개되어, 다양한 FM 성능을 자동으로 비교 분석할 수 있게 되었습니다.
- 2024년 5월: Bedrock Studio 웹 인터페이스 출시로, 조직 내 협업 기반 GenAI 애플리케이션 개발이 더욱 편리해졌습니다.
- 2025년 1월: DeepSeek-R1 모델이 Amazon Bedrock에 완전 관리형으로 추가돼 검색·분석 워크로드에 최적화된 성능을 제공하고 있습니다.

## 활용 예시

- 전자상거래: 호주 Adore Beauty는 Bedrock을 활용해 고객 맞춤형 제품 설명을 자동 생성, 마치 경험 많은 카피라이터가 24시간 대기하듯 효율을 높였습니다.
- 금융권: 대형 은행들은 내부 정책 문서 기반으로 규정 준수 챗봇을 구축, 복잡한 대출 심사 프로세스를 간결하게 안내합니다.
- 미디어: 콘텐츠 제작사는 Nova 모델을 이용해 기사 요약·이미지 생성 워크플로우를 자동화해 제작 비용을 75% 절감했습니다.

## 시작하기

1. AWS 콘솔에서 Amazon Bedrock 활성화 후 IAM 권한을 설정합니다.
2. 플레이그라운드에서 다양한 FM을 호출하며 프롬프트와 파라미터를 실험해 봅니다.
3. 애플리케이션 레벨에서 `InvokeModel` API를 호출해 프로덕션 환경에 통합합니다.
4. 필요 시 파인튜닝·RAG·에이전트 기능을 적용해 비즈니스 요구사항에 맞춰 확장합니다.

## 마치며

AWS Bedrock는 AI 모델을 ‘빌딩 블록’으로 보고, 이를 조립해 창의적 산출물을 만드는 ‘AI 레고 키트’와도 같습니다.  
필요한 기둥을 골라 쌓고, 파인튜닝이라는 접착제로 맞춤 완성작을 제작할 수 있습니다.
