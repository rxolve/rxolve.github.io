---
title: 'JetBrains 단축키 변경 방법'
date: '250611'
tags: ['Tool']
---

1. 설정 창 열기

   - Windows/Linux: `Ctrl + Alt + S`
   - macOS: `⌘ + ,`

2. 왼쪽 메뉴에서 Keymap 선택
3. 우측의 검색창에 변경할 기능명 입력 예시
   - 커서 이동 : Move Caret
   - 쿼리 실행 : Execute Statement
   - 데이터 제출 : Submit Changes
4. 해당 명령 우클릭 → Add Keyboard Shortcut 선택
5. 원하는 키 조합 입력 후 OK.

   - 복합 키를 원할 경우 ‘Second stroke’ 체크박스 사용

6. 충돌이 있을 경우 경고 팝업이 나타나며, Modify Shortcuts 버튼으로 조정할 수 있습니다.
7. 설정 완료한 다음 Apply/OK 클릭.
