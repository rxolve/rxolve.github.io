## Warp 터미널이란?

Warp는 Rust로 구현된 차세대 터미널 에뮬레이터로, macOS·Linux·Windows 환경에서 동작합니다.   
기존 쉘과 달리 블록 단위 명령 실행, IDE처럼 편집 가능한 입력창, 그리고 AI 지원 기능을 제공하여 생산성을 크게 향상시켜 줍니다.

## 주요 특징

* 모던 UX & IDE 스타일 입력 편집기

  * 커맨드를 블록 단위로 관리해 스크롤·하이라이트·복사·붙여넣기가 편리합니다.
  * 코드 에디터처럼 커서 이동, 영역 선택, 편집(이동·복사·잘라내기 등)이 가능합니다.

* AI 도구

  * 자연어 입력으로 커맨드 제안(Next Command) 및 코드 생성이 가능합니다.
  * Agent Mode에서 코드 디버깅·빌드·테스팅 작업을 AI에 위임할 수 있습니다.
  * AI 요청(AI requests)은 월별 할당량 내에서 자유롭게 사용 가능합니다.

* Warp Drive (노트북 & 워크플로우 공유)

  * Interactive Notebook과 Workflow를 만들어 팀원과 공유·협업할 수 있습니다.
  * 환경 변수 관리, 스크립트 템플릿화 등을 지원하여 표준화된 개발 환경을 구축할 수 있습니다.

* 실시간 세션 공유

  * 최대 5명의 동료와 터미널 세션을 실시간으로 공유하며 페어 프로그래밍이 가능합니다.

* 오프라인에서도 사용 가능한 기본 터미널 기능

  * ssh, git 등의 명령 실행은 물론, 컬러 출력·True Color 지원·탭 완성 등 기존 터미널 기능을 모두 지원합니다.

## 무료 플랜(Free Tier) 구성

Warp의 Free 플랜은 다음과 같은 기능을 무제한 또는 할당량 내에서 제공합니다.

1. 코어 터미널 기능 (오프라인 명령, 블록 기반 출력, 현대적 입력 편집) – 무제한
2. AI 요청

   * 월 최대 150회 무료 AI 요청
3. Next Command 제안 – 무제한
4. LLM 모델 무료 체험 (OpenAI GPT, Claude 등 선택 가능)
5. Agent Mode 코딩 기능 무료 체험
6. 개인 Warp Drive
7. 공유 Drive 내 최대 3개의 노트북 & 10개의 워크플로우
8. 실시간 세션 공유 5회
9. 커뮤니티 포럼을 통한 무료 지원

> 참고: Free 플랜은 회사 규모와 상관 없이 누구나 이용할 수 있으며, 필요시 Pro(월 \$15/연간 청구) 또는 Turbo(월 \$40/연간 청구) 플랜으로 업그레이드하여 AI 요청 한도, 세션 공유, 노트북·워크플로우 수 등을 확장할 수 있습니다.
