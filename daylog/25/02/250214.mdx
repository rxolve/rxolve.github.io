튜링의 대각선 논법은 게오르크 칸토어가 무한 집합의 크기를 비교할 때 사용했던 대각선 논법을 컴퓨터 과학에 적용한 것입니다.

이 기법의 핵심은 자기 참조와 모순 유도를 통해 특정 문제를 풀 수 없음을 보이는 것입니다.

#### 대각선 논법

튜링은 모든 프로그램이 유한한 문자열로 표현될 수 있음을 이용하여, 모든 프로그램을 일렬로 나열할 수 있다고 가정합니다.

1. 모든 프로그램을 `P1, P2, P3, ...` 처럼 번호를 매겨 나열할 수 있다고 하자.
2. 이제, 프로그램 `Pᵢ`가 입력 `I`에 대해 정지하는지 여부를 판별하는 가상의 판별기 `H(P, I)`가 있다고 가정하자.

   - `H(P, I) = True`이면 `P`가 `I`에 대해 정지한다.
   - `H(P, I) = False`이면 `P`가 `I`에 대해 무한 루프에 빠진다.

3. 이제, 특별한 프로그램 D를 설계한다.

   - D는 `H(P, P)`의 결과를 보고 반대 행동을 하는 프로그램이다.
   - 즉, 만약 `H(P, P) = True`라면 D는 무한 루프에 빠지고,
   - `H(P, P) = False`라면 D는 정지한다.

4. 그런데 D를 자기 자신에게 적용하면 (`D(D)`) 어떤 일이 벌어지는가?
   - `H(D, D) = True`이면 D는 정지해야 하지만, 설계상 무한 루프에 빠진다 → 모순!
   - `H(D, D) = False`이면 D는 멈추지 않아야 하지만, 즉시 정지한다 → 모순!

결국, 이러한 H는 존재할 수 없으므로, 정지 문제는 결정 불가능하다.

## 대각선 논법과 다른 결정 불가능성 증명

대각선 논법은 튜링의 정지 문제뿐만 아니라, 수학과 논리학에서 여러 결정 불가능성 문제를 증명하는 데 사용됩니다.

- 칸토어의 대각선 논법: 실수 집합이 자연수 집합보다 크다는 것을 보이는 증명. (실수의 개수는 가산 무한보다 큼)
- 괴델의 불완전성 정리: 수학적으로 참이지만 증명할 수 없는 명제가 존재함을 증명할 때 사용됨.
- 컴퓨터 과학에서의 결정 불가능성: 어떤 프로그램이 특정 속성을 가지는지 일반적으로 판별할 수 없음.
