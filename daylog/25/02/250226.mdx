---

Kubestronaut 프로그램은 Kubernetes 및 클라우드 네이티브 생태계에 대한 깊은 이해와 실무 경험을 갖춘 전문가임을 인증하는 일종의 명예 타이틀입니다.

CNCF(Cloud Native Computing Foundation)에서 주관하는 이 프로그램은 아래와 같은 다섯 가지 인증을 모두 성공적으로 취득해야 부여됩니다.

- Kubernetes and Cloud Native Associate (KCNA)
- Kubernetes and Cloud Security Associate (KCSA)
- Certified Kubernetes Administrator (CKA)
- Certified Kubernetes Application Developer (CKAD)
- Certified Kubernetes Security Specialist (CKS)

이 다섯 인증을 모두 활성 상태로 유지해야 Kubestronaut 자격을 얻을 수 있으며, 각 인증은 Kubernetes의 기초부터 운영, 애플리케이션 개발, 보안까지 폭넓은 영역의 전문 지식을 검증합니다.

무엇보다 멋진 재킷을 준다고 합니다. 클라우드가 필수가 된 업계에서 도전해볼만한 타이틀입니다.
