---
title: "언제 어떤 네이밍 규칙을 사용할까?"
date: "250204"
tags: ["Design"]
---

| 네이밍 규칙          | 주 사용처                                                   |
| -------------------- | ----------------------------------------------------------- |
| `camelCase`          | 변수, 함수 (JavaScript, TypeScript, Java, C#)               |
| `snake_case`         | 변수, 함수 (Python, 데이터베이스)                           |
| `kebab-case`         | URL, CSS 클래스, 파일 이름                                  |
| `PascalCase`         | 클래스, 타입, 인터페이스 (JavaScript, TypeScript, Java, C#) |
| `UPPER_SNAKE_CASE`   | 상수, 환경 변수                                             |
| `Hungarian Notation` | 레거시 코드 (거의 사용되지 않음)                            |
