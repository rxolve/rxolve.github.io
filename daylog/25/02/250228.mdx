KCNA(Kubernetes and Cloud Native Associate) 자격증은 쿠버네티스와 클라우드 네이티브 에코시스템에 대한 기초 지식을 평가하는 시험으로,

실무에서 이미 쿠버네티스를 다루고 있다면 상대적으로 진입 장벽이 낮다고 느낄 수 있습니다.

#### KCNA 자격증 기본 개요

- 시험 성격:  
  KCNA는 객관식(멀티플 초이스) 시험으로, 손으로 직접 클러스터를 구성하는 실습형 시험(CKA, CKAD 등)과는 달리 이론과 개념에 집중되어 있습니다.
- 시험 구성:
  - 문항 수: 약 60문항
  - 시험 시간: 90분
  - 합격 기준: 75점 이상
  - 유효 기간: 2년 (1회 재응시 가능)
- 출제 영역:
  1. Kubernetes Fundamentals (약 46%)
     - 쿠버네티스 아키텍처, 리소스, API, 컨테이너, 스케줄링 등
  2. Container Orchestration (약 22%)
     - 컨테이너 런타임, 네트워킹, 스토리지, 서비스 메시 등
  3. Cloud Native Architecture (약 16%)
     - 오토스케일링, 서버리스, 커뮤니티, 역할 및 거버넌스 등
  4. Cloud Native Observability (약 8%)
     - 텔레메트리, 모니터링(예: Prometheus), 비용 관리
  5. Cloud Native Application Delivery (약 8%)
     - GitOps, CI/CD 등 애플리케이션 배포 관련 내용

자세한 시험 범위와 비중은 [CNCF의 공식 커리큘럼](https://github.com/cncf/curriculum/blob/master/KCNA_Curriculum.pdf)에서 확인할 수 있습니다.
