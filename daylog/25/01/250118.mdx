---
title: "Content-Disposition: S3에 올린 파일을 다운로드만 가능하게 할 때"
date: "250118"
tags: ["HTTP"]
---

Content-Disposition은 서버가 클라이언트의 파일 처리 방식을 지정하는데 사용되는 HTTP 헤더입니다.

#### attachment

attachment를 사용하면 다운로드 가능한 파일 응답을 제공할 수 있습니다.

```http
Content-Disposition: attachment
```

#### filename

filename 속성을 통해 사용자가 저장할 파일 이름을 지정할 수 있습니다.

```http
Content-Disposition: attachment; filename="filename.jpg"
```

#### filename\*

filename\* 속성을 사용하면 UTF-8로 인코딩된 파일 이름을 지정할 수 있습니다.

한글 파일명을 지정할 때 사용합니다.

```http
Content-Disposition: attachment; filename*=UTF-8''%ED%85%8C%EC%8A%A4%ED%8A%B8.pdf

```
